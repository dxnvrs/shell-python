# Shell using python
## This is a repository for my shell project in Python 
The shell have the following features:
- Builtins (pwd, cd, type, history, echo and exit);
- the 'cd' feature handles both relative and absolute paths;
- the 'pwd' feature prints the absolute path of the current working directory to stdout;
- the shell handles single and double quotes;
- redirection feature;
- command completion;
- pipelines (dual-command, multi-command and with builtins);
- up and down arrow navigation through history;
- execution from history;
- history persistence
